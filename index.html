This is a small tool that can restart the cubeworld server if it's not responding. Just fill out the IPs (or click refresh to retrieve them automatically) and select an executable/batch file that should be run when the server is not responding. You can change the interval of the checks with the text box on the lower left (in seconds). Every x seconds, CWSRestart will check if the server process is still running and if its responding on the given IPs. If not, the server is shut down and the selected executable/batch file will run.

This application requires the .net Framework 4.5. It is available for Windows Vista, Windows 7 and Windows 8 as well as diffrent Windows Server 2008 versions.

The latest stable release is marked with Latest release. Every now and then, you'll find Pre-releases there as well. These releases are not intended for use on productive servers. However, they can be used to check out upcoming features. Please note that development builds won't notify you of updated development builds.

Github can also provide feeds for your favorite RSS reader. To stay updated on the releases, you can check this atom feed: https://github.com/ChrisK91/CWSRestart/releases.atom

If you have any questions, feel free to contact me.

If you are having trouble with CWSWeb generating error pages, try to right-click every dll-file and choose "unblock".
