Userbars


The userbars above is a static image and will not update.

Userbars are small images, that display your server status. You can include them in forums, on websites, in signatures and every other place you can think of. Userbars require statistics to be enabled. Userbars are available via http://your-server-ip:8181/Files/userbar.png.

Customization

The template that is used to create your userbar is located in the subfolder Content in the CWSWeb directory. There you will find a file called base.png and a font called font.ttf. You can replace base.png with a custom header. You can chose the size freely, however I recommend at least 25px height and 550px width. If you go below these sizes some text might get cut off. If you want to use a diffrent font, just replace the font.ttf. The gray rectangle with the text has a height of 25px.
