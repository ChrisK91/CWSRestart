Screenshots

  • CWSRestart is designed to make hosting a CubeWorld server easier
  • With under 10 clicks you can set up CWSRestart to monitor your server
  • The included UPnP wizzard can help you to forward your ports.
  • A small Man-in-the-Middle server can help you to identify users and to restrict levels - purely optional of course.
  • If you want, you can download CWSWeb to display web statistics.
  • CWSWeb also allows you to remotely manage CWSRestart,...
  • ... kick players,...
  • ... and view more advanced statistics.
